# Quiz_platform
The Quiz Platform is an interactive web application designed to facilitate digital learning and assessment through a two-role ecosystem involving Students and Faculty. The platform provides a simple environment where users can attempt multiple-choice quizzes, receive instant feedback, and view their calculated total scores upon completion.
